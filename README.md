# GoSafe – Personal Safety Walking App

GoSafe is a mobile safety application designed to increase personal security during walks, especially when users are walking alone. The app uses the smartphone’s sensors and GPS to detect unusual movements, falls, or long periods of inactivity.

If a potential emergency is detected, the app first asks the user if they are okay. If the user does not respond within a specified time, an SOS alert with the user’s location is sent to a selected emergency contact.

---

## Features

- Start a safe walk
- Real-time GPS tracking
- View current location on a map
- Walk timer
- Fall detection using phone sensors
- Inactivity detection
- Safety check notifications
- Manual SOS button
- Automatic SOS alert
- Emergency contact management
- Walk history
- Simple and clear user interface

## How the App Works

1. The user logs in using email or Google.
2. The user starts a safe walk.
3. The app monitors the user’s location and movement.
4. If a fall or unusual inactivity is detected, the user receives a safety check.
5. If the user does not respond, an SOS alert is activated.
6. The emergency contact receives a message with the user’s current location.

---

## Screens

- **Login Screen** – login with email or Google account
- **Home Screen** – access to safe walk, emergency contacts, and history
- **Walk Screen** – monitors the walk using GPS and sensors
- **Contact Screen** – add, update, or remove emergency contacts
- **History Screen** – view previous walks and safety events

---

## Design

GoSafe follows human-centered design principles. The interface is simple, clear, and easy to use, especially in stressful situations.

The color palette includes:

- Blue
- Light blue
- Light gray
- White

Blue is used as the main color because it represents trust, safety, and stability.
